Info:PussHoppingOffset

This parameter is the long DRX cycle and DRX start offset value to run onDurationTimer in DRX mode in normal status. The unit of the long DRX cycle is asub-frame. If the short DRX-Cycle is set to a value, this parameter is set to a multiple of the short DRX-Cycle. The DRX start offset is set to an integer.
